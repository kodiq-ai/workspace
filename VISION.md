# 🚀 AI Workspace — Фичи проекта (v2)

## Концепция
Десктопное приложение (Tauri v2) — красивая обёртка над нативными AI CLI (Claude Code, Gemini CLI, Codex CLI и др.) для людей без опыта программирования. Пользователи авторизуются через свои подписки, а приложение даёт им удобный визуальный интерфейс, живое превью и встроенное сообщество.

**Ключевой принцип: мы не трогаем API. Мы запускаем настоящие CLI в терминалах и красиво это показываем.**

---

## 🧠 AI через нативные CLI

### Поддерживаемые CLI
- **Claude Code** — OAuth через Anthropic
- **Gemini CLI** — OAuth через Google
- **Codex CLI (OpenAI)** — OAuth через OpenAI
- **Aider** — поддержка множества моделей
- **Ollama** — локальные модели, без подписки
- Любой новый CLI — добавляется как "профиль терминала"

### Как это работает
- Пользователь ставит CLI сам (или приложение помогает установить)
- Авторизация — через OAuth самих провайдеров (открывается браузер, логин, готово)
- Подписки — пользователь платит провайдерам напрямую, мы ничего не берём
- Приложение запускает CLI в настоящем терминале под капотом
- Вывод CLI парсится и показывается красиво в UI

### Мультимодельность
- Переключение между CLI в один клик (табы)
- Несколько CLI открыто одновременно
- Сравнение: одна задача → отправить в Claude и в Gemini → сравнить результаты рядом
- "Любимый CLI" по умолчанию для новых проектов

## 🖥️ Терминал с красивым лицом

### Умный терминал
- Настоящий терминал под капотом (полный shell-доступ)
- Красивый UI поверх: сообщения AI выделены, код с подсветкой
- Markdown-рендеринг ответов AI
- Кнопки быстрых действий: "применить", "отменить", "объясни"
- Прогресс-бар при генерации
- История команд и ответов с поиском

### Режимы отображения
- **Чат-режим** — для новичков, выглядит как мессенджер
- **Терминал-режим** — для опытных, классический терминал
- **Гибрид** — терминал внизу, красивый чат сверху

## 🖼️ Живое превью

### WebView-панель
- Правая часть экрана — полноценный WebView с работающим сайтом
- AI генерирует код → поднимается localhost → превью обновляется в реальном времени
- Кликабельные элементы — кнопки, формы, навигация работают
- Адаптивный режим — переключение: десктоп / планшет / мобилка
- Полноэкранный режим превью
- "Открыть в браузере" — для проверки в Chrome

### Визуальный инспектор (для не-программистов)
- Наведение мышки → элемент подсвечивается
- Клик на элемент → меню: "измени цвет", "сделай больше", "удали", "не работает"
- "Эта кнопка не работает" → контекст автоматически отправляется в CLI агенту
- Drag & drop элементов визуально → AI обновляет код
- Обводишь область мышкой → пишешь что хочешь изменить → AI делает
- Скриншот превью → отправить в чат комьюнити

## 👥 Сообщество (встроенная лента)

### Лента проектов
- "Смотри что я сделал" — публикация проекта прямо из Workspace
- Живое превью чужого проекта в один клик
- Лайки, комментарии, обсуждение
- "Использовать как шаблон" → чужой проект = твоя стартовая точка
- Категории: лендинги, портфолио, магазины, блоги, дашборды, игры
- Фильтры: по модели (сделано на Claude / Gemini), по популярности, по дате

### Рецепты и промпты
- Люди делятся промптами: "вот так я сделал магазин за 15 минут"
- Указание какой CLI использовался
- Рейтинг лучших промптов
- "Повтори рецепт" → промпт автоматически отправляется в твой CLI

### Шаринг из превью
- Видишь крутой элемент на чужом сайте → "как это сделано?"
- Автор может расшарить промпт конкретного элемента
- Скриншот-аннотации — обводишь область, пишешь комментарий

### Чат и общение
- Каналы по темам: #лендинги, #магазины, #помощь, #идеи, #claude, #gemini
- Превью проекта прямо в чате (не ссылка, а живой виджет)
- Личные сообщения
- Отправка кусков кода с подсветкой

### Челленджи
- Еженедельные задания: "сделай лендинг для кофейни"
- Голосование сообщества за лучшее решение
- Лидерборд и достижения / бейджи
- Сравнение: кто на каком CLI решил лучше

### Менторство и помощь
- "Позвать на помощь" → ментор видит твой проект и превью
- Ментор может предложить промпт для решения проблемы
- Система рейтинга менторов
- Видео/аудио звонки для совместной работы (опционально)
